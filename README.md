# 🎯 Betting Analytics Dashboard (Power BI)

## 📊 Overview

This project is an end-to-end betting analytics dashboard built to analyze platform performance, user behavior, and revenue trends.

It focuses on key betting KPIs such as **Stake, GGR (Gross Gaming Revenue), NGR (Net Gaming Revenue), transaction success rates, and churn analysis**.

The goal is to transform raw transactional data into actionable insights that drive business decisions.

---

## 🚀 Key Features

* 📈 Revenue tracking (Stake, GGR, NGR)
* 🌍 Regional performance analysis
* 👤 Agent & user activity insights
* 📉 Churn and retention analysis
* ⚠️ Anomaly detection in transaction trends
* 📊 Interactive dashboards for executive reporting

---

## 🛠️ Tools & Technologies

* **Power BI** (Data Visualization & Dashboarding)
* **SQL** (Data extraction & transformation)
* **Excel / Power Query** (Data cleaning & preparation)
* **DAX** (Measures & calculations)

---

## 📌 Business Questions Answered

* Which regions generate the highest revenue?
* What drives fluctuations in GGR and NGR?
* Where are transaction failures occurring?
* Which agents or products perform best?
* Are there seasonal patterns in betting behavior?
